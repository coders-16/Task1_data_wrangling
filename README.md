 **Titanic Data Wrangling Project**

 **Objective**  
This project focuses on cleaning and preparing raw Titanic dataset for analysis.

**Tasks Performed**  
\- Loaded dataset using Pandas  
\- Identified missing values  
\- Removed duplicate rows  
\- Filled missing Age values  
\- Removed Cabin column due to excessive null values  
\- Standardized column names  
\- Created new feature: age\_group

**Tools Used**  
\- Python  
\- Pandas  
\- NumPy  
\- Matplotlib  
\- Google Colab

 **Output**  
Cleaned dataset ready for analysis.
